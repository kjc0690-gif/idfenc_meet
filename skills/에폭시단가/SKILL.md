# 에폭시 도료 단가검색 스킬

- 추천단가: 아이디마켓 판매가와 웹 최저가의 중간값 (적정가)

## 작업 흐름

### 1. 사용자 요청 파악

사용자가 요청할 수 있는 유형:
- "에폭시 단가 조사해줘" → 전체 도료류 단가 검색 + 엑셀 생성
- "KCC 유니폭시 가격 검색해줘" → 특정 제품 웹 가격 검색
- "단가표 업데이트해줘" → 기존 데이터 기반으로 최신 가격 갱신
- "OO 제품 추가해줘" → 기존 단가표에 신규 제품 추가

### 2. 웹 가격 검색

WebSearch 도구를 사용하여 각 제품의 온라인 판매가를 검색한다.

검색 쿼리 예시:
- `"KCC 유니폭시 하도 ECO 14L 가격"`
- `"노루 크린폭시 코팅플러스 16kg 가격"`
- `"KCC 스포탄 하도 ECO 14kg 가격"`

검색 시 주의:
- 제조사 + 제품명 + 규격을 모두 포함하여 검색
- 최소 2~3개 판매처의 가격을 수집
- 판매처명(11번가, 쿠팡, 네이버 등)을 함께 기록
- 가격이 부가세 포함/미포함인지 확인

### 3. 추천단가 산출 로직

```
추천단가 = 아이디마켓 판매가와 웹 최저가의 중간값 (100원 단위 올림)
```

세부 규칙:
- 웹 가격만 있고 아이디마켓 가격 없음 → 웹 최저가가 추천단가
- 아이디마켓 가격만 있고 웹 가격 없음 → 아이디마켓 판매가가 추천단가
- 아이디마켓 판매가 ≤ 웹 최저가 → 아이디마켓 판매가가 추천단가 (자사가 더 저렴)
- 둘 다 있음 → 중간값 (100원 올림)
- 둘 다 없음 → '-' 표시

아이디마켓 판매가 계산:
```python
판매가 = math.ceil(원가 * 1.15 / 100) * 100
```

### 4. 엑셀 생성

`scripts/create_price_excel.py` 스크립트를 실행하여 엑셀을 생성한다. 또는 xlsx 스킬과 연계하여 openpyxl로 직접 생성할 수도 있다.

엑셀 구조:
- **시트1: 에폭시 바닥재** — KCC/노루 에폭시, 수성도료, 내화도료
- **시트2: 우레탄 방수재** — KCC 스포탄 시리즈
- **시트3: 판매점 연락처** — 주요 판매점 정보

각 시트의 열 구조 (16열):

| A | No |
| B | 제조사 |
| C | 제품명 |

| F | 아이디마켓 원가(원) |
| G | 아이디마켓 판매가(원)(+15%) |
| H-I | 웹검색① 단가/판매처 |
| J-K | 웹검색② 단가/판매처 |

| N | 추천단가(원) |
| O | 산출근거 |
| P | 비고 |

색상 규칙:

- 아이디마켓 판매가: 연한 초록 (#E2EFDA)
