# 구글 캘린더 일정 및 할 일 알림

Google Calendar MCP 도구를 사용하여 사용자의 일정을 조회하고, 로컬 할 일 파일을 관리하여, 보기 좋게 정리하여 알려주는 스킬이다.

## 알림 모드

### 1. 일간 알림 (매일 아침)
오늘부터 3일간(오늘, 내일, 모레)의 일정을 함께 보여준다. 일정이 없는 날에도 반드시 알림을 출력한다. "일정 없음"이라고 명시하고, 모든 섹션은 항상 표시한다.

**조회 방법:**
- `gcal_list_events`로 오늘 일정 조회 (timeMin: 오늘 00:00, timeMax: 오늘 23:59)
- `gcal_list_events`로 내일 일정 조회 (timeMin: 내일 00:00, timeMax: 내일 23:59)
- `gcal_list_events`로 모레 일정 조회 (timeMin: 모레 00:00, timeMax: 모레 23:59)
- **모든 캘린더**(primary, 업무, 현장스케줄, 경조사, 월례회 및 만남, 휴일)에서 조회
- timeZone은 `Asia/Seoul`을 사용
- 지난 30일간의 모든 캘린더 이벤트도 조회하여 미완료 할 일 자동 감지

**출력 형식:**
```
📅 [오늘 날짜] (요일) 일정 알림

━━━ 오늘의 일정 ━━━
• [시간] 일정명 (장소)
• 종일  일정명
(일정이 없으면)
• 오늘은 일정이 없습니다 😊

━━━ 내일 ([날짜] 요일) 일정 ━━━
• [시간] 일정명 (장소)
• 일정 없음

━━━ 모레 ([날짜] 요일) 일정 ━━━
• [시간] 일정명 (장소)
• 일정 없음

💡 오늘 하루도 화이팅!
```

**중요: 일정이 없는 날에도 이 형식을 그대로 출력한다. 오늘 일정이 0개여도 "오늘은 일정이 없습니다"라고 표시하고, 3일간 일정은 반드시 모두 보여준다. 알림을 생략하지 않는다.**

## 할 일(Todo) 관리

### 할 일 파일
할 일 목록은 `~/.claude/skills/gcal-reminder/todos.json`에 저장한다. 이 파일이 없으면 빈 배열 `[]`로 생성한다.

**파일 형식:**
```json
[
  {
    "id": 1,
    "title": "할 일 제목",
    "dueDate": "2026-03-20",
    "createdAt": "2026-03-18",
    "completed": false,
    "source": "calendar"
  }
]
```

- `dueDate`: 기한 날짜 (없으면 `null`)
- `completed`: 완료 여부 (완료되면 `true`)
- `source`: "calendar" (캘린더에서 가져온 것) 또는 "manual" (수동 추가)

### 할 일 조회 방법
매일 알림 시 아래 두 가지를 모두 확인한다:

1. **로컬 할 일 파일**: `todos.json`에서 `completed: false`인 항목 전부
2. **지난 캘린더 일정**: 모든 캘린더에서 지난 30일간의 이벤트 중, 제목에 "할 일", "TODO", "todo", "해야", "완료", "처리" 등의 키워드가 포함된 이벤트를 검색하여, 아직 `todos.json`에 없으면 자동으로 추가

### 할 일 출력 형식 (일간 알림에 포함)
일간 알림의 3일 후 일정 아래에 다음 섹션을 추가한다:

```
━━━ 미완료 할 일 ━━━
⚠️ [기한 지남 - 3/20] 보고서 제출
⚠️ [기한 지남 - 3/15] 견적서 발송
📌 [기한 없음] 사무실 정리
📌 [기한 없음] 자료 백업

📊 미완료 할 일: N개 (기한 지남: M개)
```

- 기한이 지난 할 일은 `⚠️ [기한 지남 - 날짜]` 형식으로 상단에 표시
- 기한 없는 할 일은 `📌 [기한 없음]` 형식으로 하단에 표시
- 기한이 지난 순서대로 정렬 (오래된 것이 위)
- 미완료 할 일이 없으면 "모든 할 일을 완료했습니다! 👏" 표시
- **미완료 할 일은 완료 처리할 때까지 매일 반복해서 알려준다**

### 할 일 관리 명령
사용자가 할 일을 관리할 때 아래와 같이 처리한다:

- **"할 일 추가: XXX"** → `todos.json`에 새 항목 추가
- **"할 일 추가: XXX (기한: 3/25)"** → 기한 포함하여 추가
- **"XXX 완료"** 또는 **"XXX 했어"** → 해당 항목의 `completed`를 `true`로 변경
- **"할 일 목록"** → 현재 미완료 할 일 전체 표시
- **"완료된 할 일"** → 완료된 항목 표시
- **"할 일 삭제: XXX"** → 해당 항목 삭제

### 2. 주간 알림 (매주 월요일)
이번 주 월~일요일까지의 전체 일정을 요약한다.

**조회 방법:**
- `gcal_list_events`로 이번 주 월요일~일요일 일정 조회
- timeZone은 `Asia/Seoul`을 사용

**출력 형식:**
```
📋 이번 주 일정 요약 ([월요일 날짜] ~ [일요일 날짜])

🔹 월요일 ([날짜])
  • [시간] 일정명 (장소)
  • [시간] 일정명

🔹 화요일 ([날짜])
  • 일정 없음

🔹 수요일 ([날짜])
  • [시간] 일정명
  ...

📊 이번 주 요약: 총 N개 일정

━━━ 미완료 할 일 ━━━
⚠️ [기한 지남 - 3/20] 보고서 제출
📌 [기한 없음] 사무실 정리

📊 미완료 할 일: N개 (기한 지남: M개)
```

### 3. 수동 호출
사용자가 직접 일정을 물어볼 때 적절한 범위의 일정을 조회하여 보여준다.

- "오늘 일정" → 오늘 일정만
- "내일 일정" → 내일 일정만
- "이번주 일정" → 주간 형식으로
- "다음주 일정" → 다음 주 월~일
- "이번달 일정" → 이번 달 전체
- 특정 날짜 언급 시 → 해당 날짜 일정

## 출력 규칙

1. 시간은 24시간 형식 대신 오전/오후 형식 사용 (예: 오후 2:00)
2. 종일 일정은 시간 없이 "종일" 표시
3. 장소가 있으면 괄호 안에 표시
4. 일정이 없는 날은 "일정 없음"으로 표시
5. 회의 참석자가 3명 이상이면 참석자 수만 표시 (예: "참석자 5명")
6. Google Meet 링크가 있으면 표시
7. 한국어로 출력
8. 요일은 한국어로 표시 (월, 화, 수, 목, 금, 토, 일)

## 이메일 알림

자동 스케줄 알림 시, 대화창 출력과 함께 **Gmail(kjc0690@gmail.com)로 동일한 내용을 이메일로 발송**한다.

**이메일 발송 방법:**
- `scripts/send_email.py` 스크립트를 사용하여 SMTP로 직접 발송
- 실행 명령: `python ~/.claude/skills/gcal-reminder/scripts/send_email.py "kjc0690@gmail.com" "제목" "본문"`
- 수신자: kjc0690@gmail.com
- 제목 형식:
  - 일간: `📅 [날짜] 오늘의 일정 알림`
  - 주간: `📋 [날짜] 이번 주 일정 요약`
- 본문: 대화창에 출력하는 알림 내용과 동일
- 수동 호출 시에는 이메일을 보내지 않고 대화창에만 출력

## 스케줄 설정

이 스킬은 아래 스케줄로 자동 실행되도록 설정할 수 있다:
- **매일 아침 7시 (KST)**: 일간 알림 (오늘 + 3일 후) + Gmail 발송
- **매주 월요일 아침 7시 (KST)**: 주간 알림 (이번 주 전체) + Gmail 발송

스케줄 설정은 `scheduled-tasks` MCP 또는 `CronCreate` 도구를 사용한다.
