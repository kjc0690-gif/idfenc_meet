# (주)아이디에프이앤씨 AS대응 프로세스

## 역할

방수/바닥 공사 하자 AS를 **대화형**으로 접수·처리·관리하는 스킬.
사용자가 정보를 말하면 클로드가 처리한다. 사용자가 직접 입력하는 것은 최소화.

## 핵심 경로

| 구분 | 경로 |
|------|------|
| **프로젝트 폴더** | `G:\내 드라이브\클로드작업기록\이메일발송\waterproof-as-sync\` |
| **AS 데이터** | `G:\내 드라이브\클로드작업기록\영업자료\AS\` |
| **AS 사진** | `G:\내 드라이브\클로드작업기록\영업자료\AS\사진\` |
| **엑셀 생성** | `create_excel.py` |

## 대화형 운영 방식

### 원칙
- **클로드가 하는 것**: 데이터 입력, 엑셀 생성, 메일 발송, 캘린더 등록, 백업
- **사용자가 하는 것**: 정보를 말로 전달, 사진을 폴더에 넣기, 최종 승인

### 사용 예시

사용자: "AS 접수해줘. 삼성전자 평택캠퍼스, 지하주차장 바닥 에폭시 들뜸. 담당자 이과장 010-1234-5678"
→ 클로드가 자동으로:
  1. AS 케이스 생성 (JSON)
  2. 접수 확인서 표시 → 승인 요청
  3. 승인 후 저장

사용자: "현장 방문 일정 잡아줘. 다음주 화요일"
→ 클로드가:
  1. 캘린더에 일정 등록
  2. 담당자에게 메일 발송 (승인 후)

사용자: "사진 폴더에 넣었어"
→ 클로드가:
  1. 사진 폴더 확인
  2. AS 케이스에 사진 연결

## AS 처리 단계별 대화 흐름

### 1단계: AS 접수 - "AS 접수해줘"

클로드가 순서대로 물어볼 것:
1. **현장명** - "어느 현장인가요?"
2. **하자 위치** - "하자 위치가 어디인가요?" (예: 지하1층 주차장)
3. **하자 내용** - "어떤 하자인가요?" (예: 바닥 에폭시 들뜸)
4. **접수 경로** - "어떻게 접수됐나요?" (전화/메일/현장발견)
5. **고객 담당자** - "담당자 이름과 연락처?"
6. **긴급도** - "긴급인가요, 일반인가요?"

한번에 여러 정보를 말하면 추가 질문 없이 바로 처리.

접수 완료 시 표시:
```
═══════════════════════════════
  AS 접수 확인
═══════════════════════════════
케이스 ID: AS20260403-001
현장명: 삼성전자 평택캠퍼스
하자 위치: 지하1층 주차장
하자 내용: 바닥 에폭시 들뜸
접수 경로: 전화
담당자: 이과장 (010-1234-5678)
긴급도: 일반
접수일: 2026-04-03
═══════════════════════════════
접수 진행할까요? (Y/N)
```

### 2단계: 현장 방문 - "현장 방문 등록해줘"

클로드가 물어볼 것:
1. **방문 일시** - "언제 방문하나요?"
2. **방문자** - "누가 가나요?"
3. **준비물** - 하자 내용 기반 자동 추천

클로드가 자동으로:
- 구글 캘린더에 일정 등록
- 담당자에게 방문 안내 메일 발송 (승인 후)

### 3단계: 시공 - "시공일지 작성해줘"

클로드가 물어볼 것:
1. **작업 내용** - "오늘 뭐 했나요?"
2. **사용 자재** - "자재 뭐 썼나요?"
3. **인원** - "몇 명이 작업했나요?"
4. **진행률** - "몇 % 완료?"
5. **사진** - "사진 폴더에 넣었나요?"

### 4단계: 완료 - "AS 완료 처리해줘"

클로드가 자동으로:
- 완료 보고서 엑셀 생성
- 고객에게 완료 안내 메일 발송 (승인 후)
- 구글드라이브 백업

## AS 데이터 저장 구조

```
G:\내 드라이브\클로드작업기록\영업자료\AS\
├── cases.json              ← 전체 AS 케이스 목록
├── AS20260403-001\         ← 케이스별 폴더
│   ├── 기본정보.json
│   ├── 방문기록.json
│   ├── 시공일지.json
│   ├── 완료보고.json
│   └── 사진\
│       ├── 접수_001.jpg
│       └── 시공_001.jpg
└── AS20260403-002\
    └── ...
```

## AS 현황 조회

사용자: "AS 현황 보여줘"
→ 클로드가 cases.json 읽어서 표로 표시:

```
| 케이스 ID | 현장명 | 하자내용 | 상태 | 접수일 |
|-----------|--------|----------|------|--------|
| AS20260403-001 | 삼성전자 평택 | 에폭시 들뜸 | 접수 | 04-03 |
| AS20260402-001 | 포스코 송도 | 방수 누수 | 시공중 | 04-02 |
```
