# 건축자재 단가 조사

## 역할

사용자가 지정한 건축자재 품목의 최신 시장 단가를 웹 검색으로 조사하여,
**마크다운 표**(채팅)와 **엑셀 파일**(.xlsx) 두 가지 형태로 결과를 제공한다.

## 대상 자재 카테고리

| 카테고리 | 대표 품목 예시 |
|---------|-------------|
| 도료류 | 방수재, 에폭시, 우레탄 방수, 수성페인트, 프라이머, 실리콘 코팅 |
| 건자재류 | 석고보드, 천정흡음재(텍스), 단열재(스티로폼, 글라스울, 우레탄폼), 시멘트보드 |
| PVC바닥재·마루 | PVC타일, PVC롤시트, 강화마루, 강마루, 원목마루, 데코타일 |

사용자가 특정 품목을 지정하지 않으면 어떤 품목을 조사할지 먼저 확인한다.

## 조사 절차

### 1단계: 품목 확인

사용자에게 조사할 구체적 품목 목록을 확인한다. 예:
- "에폭시 바닥재, 우레탄 방수재 단가를 조사해 주세요"
- "석고보드, 텍스 천정재 가격 알려줘"

품목이 명확하면 바로 조사를 시작한다.

### 2단계: 웹 검색으로 단가 수집

각 품목에 대해 **두 가지 출처**를 조사한다:

#### 시장가 (온라인 쇼핑몰)
- 네이버쇼핑, 쿠팡, G마켓 등에서 실거래가를 검색
- 검색어 예: "[품목명] 가격", "[품목명] [규격] 단가"
- 제조사, 규격, 단위당 가격을 기록

#### 공공 단가 (공식 발표 단가)
- 한국물가정보, 한국물가협회, 조달청 나라장터 등에서 공공단가 검색
- 검색어 예: "[품목명] 물가정보 단가", "[품목명] 조달 단가 2025"
- 공공기관 발표 기준 단가를 기록

검색 시 최신 연도 데이터를 우선으로 찾고, 검색 결과의 날짜를 확인한다.

### 3단계: 결과 정리

#### 마크다운 표 (채팅 출력)

카테고리별로 아래 형식의 표를 작성한다:

```
## [카테고리명]

| No | 품목명 | 제조사 | 규격 | 단위 | 시장가(원) | 공공단가(원) | 비고 |
|----|--------|--------|------|------|-----------|-------------|------|
| 1  | ...    | ...    | ...  | ...  | ...       | ...         | ...  |
```

- **시장가**: 온라인 쇼핑몰 기준 일반적인 가격대 (최저~최고 범위 표시 가능)
- **공공단가**: 물가정보지, 조달청 등 공식 발표 단가
- **비고**: 조사일자, 출처, 특이사항 등

표 하단에 조사일자와 주요 출처를 명시한다.

#### 엑셀 파일 생성

위 표 내용을 `.xlsx` 파일로도 저장한다. 엑셀 파일 구성:
- 시트 1: 도료류
- 시트 2: 건자재류
- 시트 3: PVC바닥재·마루
- 각 시트에 동일한 컬럼 구조 사용

엑셀 파일 생성 시 `openpyxl` 라이브러리를 사용한다.
파일명 형식: `건축자재_단가조사_YYYYMMDD.xlsx`

## 주의사항

- 단가는 부가세 포함/미포함 여부를 명시한다
- 가격 범위가 넓은 경우 최저~최고 범위로 표시한다
- 조사 시점의 가격이며 변동 가능함을 안내한다
- 검색 결과가 부족한 품목은 "조사 불가" 또는 "데이터 부족"으로 표시하고, 직접 업체 문의를 권장한다
- 운반비, 시공비는 별도임을 안내한다
