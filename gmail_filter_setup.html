Gmail 필터 자동 설정

아래 링크를 클릭하면 Gmail 필터 생성 폼이 열립니다.

1. 프로모션/광고 자동 삭제 필터

Trip.com Alibaba Adobe

프로모션 필터 만들기
폼이 열리면:
1. "필터 만들기" 클릭
2. "삭제" 체크
3. "일치하는 대화에도 필터 적용" 체크
4. "필터 만들기" 클릭

2. 소셜 알림 자동 삭제 필터

Facebook 친구추천 Facebook 알림 Instagram

소셜 필터 만들기
폼이 열리면:
1. "필터 만들기" 클릭
2. "삭제" 체크
3. "일치하는 대화에도 필터 적용" 체크
4. "필터 만들기" 클릭

3. (선택) 필터 XML 가져오기

위 2개 필터를 한번에 가져올 수도 있습니다.

Gmail 필터 설정 열기
설정 페이지에서:
1. "필터 가져오기" 클릭
2. "파일 선택" → gmail_filters.xml 선택
3. "파일 열기" → "필터 만들기" 클릭
