# 2026-03-24 작업기록: 구글 캘린더 일정 및 할 일 알림 스킬 구축

## 작업 개요
Google Calendar MCP와 Gmail SMTP를 연동하여, 매일 아침 3일간 일정 + 미완료 할 일을 스마트폰으로 알림받는 시스템을 구축했다.

## 작업 내용

### 1. gcal-reminder 스킬 생성
- **파일**: `~/.claude/skills/gcal-reminder/SKILL.md`
- 일간 알림: 오늘, 내일, 모레 3일간 일정 조회
- 주간 알림: 매주 월요일 이번 주 전체 일정 요약
- 수동 호출: 사용자가 일정 관련 키워드 입력 시 자동 트리거

### 2. 6개 캘린더 통합 조회
- primary (kjc0690@gmail.com)
- 업무
- 현장스케줄
- 경조사
- 월례회 및 만남
- 대한민국 휴일

### 3. 할 일(Todo) 관리 시스템
- **파일**: `~/.claude/skills/gcal-reminder/todos.json`
- 캘린더에서 할 일 키워드 자동 감지 후 추가
- 기한 지남/기한 없음 구분 표시
- 미완료 할 일은 완료 처리할 때까지 매일 반복 알림
- 할 일 추가/완료/삭제 명령 지원

### 4. Gmail SMTP 이메일 발송
- **파일**: `~/.claude/skills/gcal-reminder/scripts/send_email.py`
- Gmail 앱 비밀번호로 SMTP 직접 발송
- 스마트폰 Gmail 앱으로 푸시 알림 수신 가능

### 5. 자동 스케줄 설정
- **scheduled-tasks**: `gcal-daily-reminder`
- 매일 아침 7시(KST) 자동 실행
- 월요일은 주간 알림으로 자동 전환

## 출력 형식 예시
```
📅 2026년 3월 24일 (화) 일정 알림

━━━ 오늘의 일정 (3/24 화) ━━━
• 오늘은 일정이 없습니다 😊

━━━ 내일 (3/25 수) 일정 ━━━
• 종일  와이프생일 🎂
• 오전 8:00  와이프생일 [!]

━━━ 모레 (3/26 목) 일정 ━━━
• 일정 없음

━━━ 미완료 할 일 ━━━
⚠️ [기한 지남 - 3/4] 온양AS
⚠️ [기한 지남 - 3/13] 밤캘케리어 AS접수

📊 미완료 할 일: 2개 (기한 지남: 2개)

💡 오늘 하루도 화이팅!
```

## 수정 이력
1. 초기 구축: 오늘 + 3일 후 일정 알림
2. 내일 일정 추가: 오늘 + 내일 + 3일 후
3. 모든 캘린더 통합: 6개 캘린더 전체 조회
4. 미완료 할 일 추가: 기한 지남 할 일 매일 반복 알림
5. 3일간 수정: "3일 후" → "오늘, 내일, 모레" 3일간으로 변경
6. 이메일 발송: Gmail SMTP로 스마트폰 알림 구현
