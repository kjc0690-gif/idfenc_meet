## 저장 규칙
- 회의록 저장 위치: TBM/회의록/ 폴더 안에 날짜별 파일로 저장
- 파일명 형식: TBM_회의록_20260318.xlsx
- 날짜별 폴더를 따로 만들지 않음

## 입력 순서
1. 참석자 (쉼표로 구분 입력)
2. 현장명
3. 소속(회사명)
4. 작업명
5. TBM 장소
6. 작업내용 (사용자가 직접 입력)
- 진행자 입력은 없음 (참석자 첫 번째가 리더)

## 자동 생성 항목
- 날짜: 오늘 날짜 자동
- 위험요인 3개: 작업명/작업내용 키워드로 템플릿에서 자동 생성
- 대책 3개: 위험요인에 맞는 대책 자동 생성
- 중점위험요인 선정/대책: 자동 생성
- 참석 인원수: 참석자 수 자동 계산

## 사진 처리
- 사진 폴더를 선택하면 오늘 날짜 사진만 자동으로 골라서 넣음
- 날짜 판별: 파일명, 파일 수정일, EXIF 촬영일 확인
- 사진은 원본 그대로 유지 (리사이즈 하지 않음)

## 사용자 주의사항
- 사용자는 코딩을 모르는 초등학생 수준으로 설명할 것
- 전문 용어 사용 금지 (터미널, 환경변수 등)
- 단계별로 쉽게 안내할 것
