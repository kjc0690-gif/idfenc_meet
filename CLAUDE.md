# TBM 자동화 프로젝트

## 프로그램 위치 (Google Drive)
- 기본 경로: `G:\My Drive\업무자동화\`
- 프로그램: `G:\My Drive\업무자동화\TBM\tbm_recorder.py`
- 템플릿 파일: `G:\My Drive\업무자동화\TBM\20260318\TBM_회의록_20260316.xlsx` (이 엑셀 양식을 그대로 복사해서 사용. 절대 새로 만들지 말 것)
- 회의록 저장: `G:\My Drive\업무자동화\TBM\회의록\` 폴더에 날짜별 파일명으로 저장 (폴더를 날짜별로 만들지 않음)

## 핵심 규칙
- 템플릿 엑셀 파일을 복사 → 데이터만 채워넣는 방식
- 진행자 입력 없음
- 작업내용은 사용자가 입력
- 위험요인, 대책, 중점위험요인, 중점대책은 프로그램이 자동 생성 (키워드 템플릿 방식)
- 사진 폴더 지정 시 오늘 날짜 사진만 자동 선택 (파일명, 수정일, EXIF 확인)
- 사진은 원본 유지

## 엑셀 셀 위치 (TBM 회의록 시트)
- B3: TBM 일시
- B4: 현장명 + 작업명
- B5: 작업내용
- B6: TBM장소
- A8~A10: 위험요인 ①②③
- D8~D10: 대책 ①②③
- C11: 중점위험요인 선정
- C12: 중점위험요인 대책
- A13: TBM 리더확인
- A16~A18: 안전조치 확인 (위험요인 반복)
- A24~: 참석자 이름 (3열 배치)

## 엑셀 셀 위치 (사진대지 시트)
- A2: 현장명
- A3: 작업명
- A4: 날짜
- B6: 공종
- B7: 위치
- A5: 큰 사진 (첫 번째)
- A8: 왼쪽 아래 사진 (두 번째)
- C8: 오른쪽 아래 사진 (세 번째)
